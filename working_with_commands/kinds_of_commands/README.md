### 命令的种类

一条命令不外乎以下4种情况。

- 可执行程序。
    可执行程序就像在/usr/bin目录里看到的所有文件一样。
    在该程序类别中，程序可以编译为二进制文件，比如C、C++语言编写的程序，也可以是shell、Perl、Python、Ruby等脚本语言编写的程序。

- shell内置命令。
    bash支持许多在内部称之为shell builtin的内置命令。例如，cd命令就是shell内置指令。

- shell函数。
    shell函数是合并到环境变量中的小型shell脚本。

- alias命令。
    我们可以在其他命令的基础上定义自己的命令。
